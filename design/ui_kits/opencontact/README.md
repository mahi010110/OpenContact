# UI kit — OpenContact (app)

Recréation de la vue principale « Pistes » dans la nouvelle identité « Utilitaire 98 » :
barre de menu système, barre d'outils, liste de pistes + zone carte, barre d'état, modale d'ajout.

Interactif : bascule clair/sombre, changement de vue, sélection d'une piste (sélection marine), ouverture de la modale « Ajouter une piste ».

La zone carte est un **placeholder** (grille + pins carrés) — en production, le fond CARTO/Leaflet reprend ces pins carrés et la légende-fenêtre.
