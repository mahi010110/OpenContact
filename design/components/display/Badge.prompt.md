Badge pixel en Silkscreen 8px, capitales — les étiquettes système d'OpenContact (PARTAGÉ, PRIVÉ, statuts).

```jsx
<Badge tone="shared">Partagé</Badge>
<Badge tone="private">Privé</Badge>
<Badge tone="warn">Relance due</Badge>
```

Le ton `private` porte la trame dither — le privé est visuellement « grillagé ».
