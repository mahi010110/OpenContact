Champ de saisie « en creux » : fond champ, bordure encre 1px, léger biseau inversé.

```jsx
<Input placeholder="Ex : Orange Cyberdefense" />
<Input multiline placeholder="Notes perso…" />
<Input mono placeholder="https://…" />
```

`multiline` → textarea ; `mono` → IBM Plex Mono pour URLs et valeurs techniques.
